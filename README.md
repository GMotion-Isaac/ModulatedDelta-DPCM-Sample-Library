# About this library
Modulated Delta is a non-profit, royalty-free DPCM sample library under the Unlicense license. Anyone, including you, can share and download DPCM samples uploaded by other users to use in 8-bit musical composing applications.

## How to submit your own DPCM Sample
In this GitHub repository, go to the **Pull Requests** tab and initiate a new Pull Request (PR). Your PR must include the `.wav` or `.dcm` file, which will then be reviewed by library contributors. Your DPCM sample must abide by the following criteria:

### Criteria to get a DPCM Sample added
1. Your sample's sample rate must be less than or equal to 32000 Hz (32.0 kHz).
2. Your sample must be entirely made by yourself (synthesized or recorded from real-life sounds) and completely non-copyrighted.
3. Your `.dcm` file must not exceed **4,081 bytes**, but we recommend aiming for under **2,000 bytes**. *(Note: This size limit does not apply to submitted `.wav` files, as their raw header data makes them larger before they are converted/compressed by the algorithm).*
4. Your sample must be properly trimmed to eliminate static. *(Note: This does not apply to `.wav` files, as they will be manually trimmed during the review process).*
5. Your sample should follow this name formatting: `MDUserGenerated_[yourusername]_[category][presetnumber]_[rootnote]`, for example `MDUserGenerated_CoolDuck645_PunchyBassSlap08_A#.dcm`. If your sample's name does not follow this formatting, the reviewing process will rename it back to the formatting type. Please make sure your username doesn't include sensitive data like real names or it will be replaced with your MD Library ID. You can also use aliases.
#### Important note when uploading your DPCM Samples.
A new folder will be created with your MD Library ID and for each sample category with the preset number that is uploaded a new folder inside your MD Library ID will be added.
