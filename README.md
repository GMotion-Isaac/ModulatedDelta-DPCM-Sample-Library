# ModulatedDelta: DPCM Sample Library
In this library, you will find lots of DPCM samples as .dcm files.
## How to request your own DPCM samples to be put on this library:
In this GitHub repo, go to the Pull Requests tab and create a new pull request. Make sure to include your .dcm or .wav file so it can be added.
## How to put DPCM samples on FamiStudio
In the FamiStudio app, look at the left sidebar, and click the folder icon where it says "DPCM Samples". Find your .dcm file and find the directory where you placed the .DCM file and upload that .DCM file.
