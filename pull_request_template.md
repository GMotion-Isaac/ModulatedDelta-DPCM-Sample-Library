## PR Checklist
Make sure your sample(s) follows all the criteria before opening your Pull Request. Check the boxes once you verify them:

- [ ] **Sample Rate:** My sample rate is less than or equal to 32000 Hz (32.0 kHz).
- [ ] **Copyright:** This sample was made entirely by myself and is completely non-copyrighted.
- [ ] **File Size:** My `.dcm` file does not exceed 4,081 bytes (preferably under 2,000 bytes) OR I am uploading a `.wav` file to be compressed by the algorithm.
- [ ] **Trimming:** The sample is properly trimmed to eliminate static.
- [ ] **Naming Convention:** My file follows this formatting: `MDUserGenerated_[yourusername]_[category][presetnumber]_[rootnote]`
